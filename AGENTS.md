# AGENTS.md


## 前提


-このリポジトリはDjangoの簡易アプリです。


## 作業手順


1. 自分にアサインされた Issue を確認する
2. Issueの内容に応じたブランチを作成する
3. Issueに必要な変更を行う
4. 細かい単位でコミットする
5. ブランチをプッシュする
6. プルリクエストを作成する


## Git ルール


- 上記の作業手順に従って作業してください。
- 'main'に直接コミットしないでください。
- コミットは細かい単位で行ってください。
- コミットメッセージは日本語で書いてください。


## プルリクエスト


- プルリクエストには日本語でコメントをつけてください。
